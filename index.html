Project Overview

The Online Pharmacy project is a web application that simplifies the process of purchasing medications and healthcare products. It connects users with verified pharmacies and offers seamless online ordering, prescription uploads, and doorstep delivery services.

Features

  • User Registration and Login
  • Search and Filter Medications
  • Upload Prescriptions
  • Secure Payment Gateway
  • Order Tracking and Notifications

Technologies Used

This project utilizes a robust tech stack to ensure scalability and reliability:

  • Frontend: HTML, CSS, JavaScript, React.js
  • Backend: Node.js, Express.js
  • Database: MongoDB
  • Authentication: JWT (JSON Web Tokens)
  • Payment Integration: Stripe API
